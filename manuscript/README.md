# Manuscript Notes

This folder is intended to hold the manuscript source files associated with the revised NeuroCypher ASD paper.

Currently included:

- `springer_revision_assets/`: the manuscript-related files that were already present in the project workspace, including `sn-article.tex`, bibliography/style files, table `.tex` files, and manuscript figures.

Important:

- The paper text shared in the working discussion may be newer than the manuscript source already stored in the original project folder.
- Before pushing this repository to GitHub as the paper companion repository, verify that the exact submission draft you want readers to see is present here.
- If your latest Springer submission `.tex` file is maintained outside the project workspace, copy it into this folder before publishing the repository.
